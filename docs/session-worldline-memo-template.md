# 世界線メモフォーマット

## 概要

1on1テストセッション終了後にユーザーに送付する「世界線メモ」のフォーマットです。このメモは、セッションで比較した世界線と指標差分を整理し、ユーザーの意思決定を支援するための参考資料として提供します。また、匿名化した上で「Evidence Pack」にも流用します。

### 3. 指標差分

以下の3つの主要指標で、各世界線を比較しました。

#### 指標1: 40〜50代のキャッシュフロー余裕（月あたり）

| 世界線 | 40〜50代のCF余裕 | 備考 |
|---|---|---|
| A: 賃貸継続 | +15万円/月 | 住宅ローンがないため余裕が大きい |
| B: 6,000万購入 | +8万円/月 | ローン返済により余裕が減少 |
| C: 8,000万購入 + 駐在 | +12万円/月 | 駐在中の賃貸収入により余裕が回復 |

**解釈:**
- 賃貸継続（A）が最も余裕がある
- 購入する場合、海外駐在の賃貸収入（C）が余裕を補う

#### 指標2: 60歳時点の資産残高

| 世界線 | 60歳時点の資産 | 備考 |
|---|---|---|
| A: 賃貸継続 | 8,500万円 | 資産運用が継続されるため高い |
| B: 6,000万購入 | 6,200万円 | 頭金とローン返済により資産が減少 |
| C: 8,000万購入 + 駐在 | 7,800万円 | 賃貸収入により資産が回復 |

**解釈:**
- 賃貸継続（A）が最も資産が多い
- 購入する場合、海外駐在の賃貸収入（C）が資産を補う

#### 指標3: 取り崩し開始年齢

| 世界線 | 取り崩し開始年齢 | 備考 |
|---|---|---|
| A: 賃貸継続 | 68歳 | 資産が多いため取り崩し開始が遅い |
| B: 6,000万購入 | 72歳 | ローン完済後は住宅費がゼロになるため取り崩し開始が遅い |
| C: 8,000万購入 + 駐在 | 70歳 | Bより資産が多いため取り崩し開始が早まる |

**解釈:**
- 購入する場合、ローン完済後は住宅費がゼロになるため、取り崩し開始が遅くなる
- 賃貸継続（A）は資産が多いが、賃貸費用が続くため取り崩し開始が早まる

### 5. 次のステップ

このメモを参考に、以下のステップで意思決定を進めてください。

1. **世界線の再検討**
   - 今回比較した3つの世界線以外に、検討すべき選択肢があるか考える
   - 例：7,000万円の中間案、海外駐在なしのケースなど

2. **前提の見直し**
   - 年収・資産・ライフイベントの前提が変わった場合、再度計算する
   - 例：転職、子どもの増加、親の介護など

3. **追加の指標**
   - 今回の3つの指標以外に、重視したい指標があるか考える
   - 例：教育費の余裕、介護費用の準備、早期リタイアの可能性など

4. **パートナーとの共有**
   - このメモをパートナーと共有し、一緒に検討する
   - 意見が分かれる場合は、優先順位を明確にする

5. **再セッションの検討**
   - 前提が大きく変わった場合や、追加の世界線を比較したい場合は、再セッションを検討する

## フォーマットの使い方

### 1. セッション中にメモを取る

- Google Docs / Notion / Obsidian などでこのフォーマットをコピー
- セッション中に、ユーザーの回答を記録
- 世界線と指標差分を計算しながら埋めていく

### 2. セッション後に整理

- セッション終了後、メモを整理
- 匿名化が必要な箇所（年齢・勤務先・物件名など）を削除またはレンジ表記に変換
- インサイトを1〜3個に絞る

### 3. ユーザーに送付

- PDF / Google Docs / Markdown のいずれかで送付
- メールに添付するか、共有リンクを送る

### 4. Evidence Packに流用

- 匿名化済みのメモを元に、Evidence Pack用のケース例を作成
- 3件のセッションが溜まったタイミングで、LP v5のトップページに掲載
