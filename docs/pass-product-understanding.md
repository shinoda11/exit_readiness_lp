# Pass商品（29,800円）の理解

## 実際のツール確認結果

URL: https://exit-readiness-os.vercel.app/

## Pass商品で提供している機能

### 1. Asset Cockpit（資産コックピット）
**目的：** 現在の収入・支出・資産から、「何歳まで働けば資産が尽きないか」と「どのレバー（収入・支出・退職年齢・ライフイベント）をどう動かすか」を一枚で把握する

**主要機能：**
- **世帯サマリー** - 年収、税・社保、手取り、生活費、貯蓄余力、貯蓄率を表示
- **基本情報入力** - 現在の年齢、引退予定年齢、世帯収入、支出、住宅ローン、現在資産
- **詳細チューニング** - RSU、副業収入、想定利回り、退職後設定、インフレ率、実効税率
- **プラン切替** - 1人プラン / 夫婦（2馬力）プラン

### 2. Exit Readiness Score（67点など）
**4つの指標：**
- **Survival（56）** - 長期サバイバビリティ（成功確率）
- **Lifestyle（66）** - ライフスタイル余力
- **Risk（80）** - ポートフォリオリスク
- **Liquidity（80）** - ロック資産のバランス

**判定結果：**
- 25倍ライン到達年齢（51歳など）
- 100歳時点資産（1.55億など）
- 生涯投資総額（15.6千万など）

### 3. 資産推移シミュレーション
**4つのビュー：**
- **全体ビュー** - 金融資産と総資産の推移
- **Rent vs Buy** - 賃貸 vs 購入の比較
- **金融資産** - 流動資産のみ
- **総資産** - 流動資産 + DCロック資産 + 自宅の純資産

**グラフ表示：**
- 35歳〜100歳までの資産推移
- FIREライン（資産＝年間支出×25倍）の表示
- 中央値シナリオと成功確率の可視化

### 4. 年間キャッシュフロー
**表示内容：**
- イベント込み収支
- ベース収支（イベント除く）
- 資産取り崩し（赤字補填）
- 年齢別の詳細キャッシュフロー表

### 5. 戦略アドバイス
**提供内容：**
- NEXT ACTIONS（次の一手）
- Exit の安全度評価
- 将来のライフイベントを含めた余白評価
- 具体的な改善提案（退職後のパートタイム収入確保など）

### 6. 目標設定（GOAL LENS）
**3つの質問：**
- **Q1 働き方のゴール** - A: 65〜70まで働ければOK / B: 50前後でフルタイムを降りたい / C: フルFIREも視野
- **Q2 残し方のスタンス（遺産観）** - L0: こだわりなし / L1: すこし残したい / L2: しっかり残したい
- **Q3 大事にしたいライフイベント（最大3）** - 海外留学、世界一周、都心マンション購入、海外/地方二拠点、キャリアチェンジ、早めのセミリタイア、子どもの教育費を手厚く

## 京都モデルv0.3との対応関係

### 現在のツール（Vercel版）の構成
1. **Timeline Overview** - 時系列ビュー
2. **Dashboard** - ダッシュボード（Asset Cockpit）
3. **Simulator** - シミュレーター
4. **Comp（RSU）** - RSU管理

### 京都モデルv0.3で実装すべき構成
1. **Pass Onboarding 3タスク** - compare_viewed, lever_changed, memo_generated
2. **Cockpit** - シナリオ固定3本（Rent/Buy/Buy+Shock）、レバー4つ（物件価格/頭金/投資入金/ショック選択）
3. **Decision Memo** - 3ブロック固定（上限レンジ/3シナリオ結論/次の30日アクション）

## Pass商品の価値提案

**29,800円で提供する価値：**
- 「何歳まで働けば資産が尽きないか」を可視化
- 「どのレバーをどう動かすか」を具体的に示す
- Exit Readiness Scoreで現在地を数値化
- 資産推移シミュレーションで将来を予測
- 戦略アドバイスで次の一手を提案

**ターゲットユーザー：**
- 世帯年収1,500万円以上
- 金融資産あり
- 数字入力OK
- 大きな選択ぜんぶ（6,000万〜8,000万クラスの"人生の決断"）を世界線で比べたい

## 実装上の注意点

1. **シナリオ固定3本** - Rent/Buy/Buy+Shockは、現在のツールの「Rent vs Buy」ビューに対応
2. **レバー4つ** - 物件価格/頭金/投資入金/ショック選択は、現在のツールの「詳細チューニング」に対応
3. **Decision Memo 3ブロック** - 上限レンジ/3シナリオ結論/次の30日アクションは、現在のツールの「戦略アドバイス」と「NEXT ACTIONS」に対応
